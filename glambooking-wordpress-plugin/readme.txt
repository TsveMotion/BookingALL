== Description ==

**GlamBooking WordPress Integration** allows you to seamlessly embed your GlamBooking booking system into your WordPress website. View bookings, analytics, and manage appointments directly from your WordPress dashboard.

### Features

* **Booking Widget Embed** - Add booking forms to any page using shortcodes, Elementor, or Gutenberg
* **WordPress Dashboard Integration** - View bookings and analytics inside WordPress admin
* **Elementor Support** - Drag-and-drop booking widget for Elementor page builder
* **Gutenberg Block** - Native WordPress block editor integration
* **Secure API Authentication** - Enterprise-grade security with API keys
* **Universal Business Support** - Works for all business types: beauticians, barbers, hairdressers, nail techs, spas, and more

### Requirements

* **GlamBooking PRO Account** - WordPress integration is exclusive to PRO plan members
* Active GlamBooking account at [beauticians.glambooking.co.uk](https://beauticians.glambooking.co.uk)

### How It Works

1. Install and activate the plugin
2. Get your API keys from GlamBooking Settings → WordPress Integration
3. Enter your API keys in WordPress Settings
4. Add booking widgets using shortcode, Elementor, or Gutenberg
5. View bookings directly in WordPress admin

### Shortcode Usage

Basic shortcode:
`[glambooking business="YOUR_BUSINESS_ID"]`

With optional parameters:
`[glambooking business="123" service="facials" theme="minimal" height="600px"]`

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/glambooking-wordpress/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to GlamBooking → Settings and enter your API credentials
4. Start embedding booking widgets!

== Frequently Asked Questions ==

= Do I need a GlamBooking account? =

Yes, you need an active GlamBooking PRO account. WordPress integration is only available for PRO plan members.

= How do I get API keys? =

1. Log in to your GlamBooking dashboard
2. Navigate to Settings → WordPress Integration
3. Click "Generate API Keys"
4. Copy and paste them into WordPress

= Does this work with Elementor? =

Yes! Search for "GlamBooking" in the Elementor widget panel and drag it onto your page.

= Does this work with Gutenberg? =

Yes! Click the "+" button in the editor and search for "GlamBooking" to add the booking block.

= Can I customize the booking widget? =

Yes, you can customize the theme, height, and pre-select services using shortcode parameters or widget settings.

= Is my data secure? =

Absolutely. All API communication is encrypted with HTTPS and uses secure authentication tokens. Your secret key is never exposed to clients.

= What business types are supported? =

All business types are supported: beauticians, barbers, hairdressers, nail techs, spas, massage, lash artists, PMU, skincare, and more.
